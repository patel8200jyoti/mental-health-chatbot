MongoDB```**Chat pipeline:**```Message ‚Üí load history from MongoDB ‚Üí [SystemMessage + history + HumanMessage]        ‚Üí asyncio.gather(detect_crisis(), model.ainvoke())        ‚Üí save messages ‚Üí crisis? append helplines : return response```**Journal save:**```Content ‚Üí analyse_journal() via LangChain ‚Üí Llama returns {mood, sentiment, reflection}        ‚Üí saved to MongoDB alongside journal content```**Crisis detection (2 stages, runs concurrently with chat):**```Stage 1 ‚Üí keyword scan (15 keywords, instant)Stage 2 ‚Üí LLM risk check if no keyword (none/mild/moderate/severe)Crisis  ‚Üí saved to crisis_alerts + helplines appended to response```---## Collections`users` ¬∑ `chats` ¬∑ `chat_messages` ¬∑ `moods` ¬∑ `journals` ¬∑ `professional_links`
